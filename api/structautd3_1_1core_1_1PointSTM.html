Detailed Description

PointSTM provides a function to display the focus sequentially and periodically.

PointSTM uses a timer on the FPGA to ensure that the focus is precisely timed. PointSTM currently has the following three limitations.

  1. The maximum number of control points is driver::POINT_STM_BUF_SIZE_MAX.
  2. Only a single focus can be displayed at a certain moment.
