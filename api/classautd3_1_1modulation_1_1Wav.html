[in]mod_sampling_freq_divsampling frequency of the Modulation

The sampling frequency of AUTD is shown in autd::MOD_SAMPLING_FREQ, and it is not possible to modulate beyond the Nyquist frequency. No modulation beyond the Nyquist frequency can be produced. If samplingFreq is less than the Nyquist frequency, the data will be up-sampled. The maximum modulation buffer size is shown in autd::MOD_BUF_SIZE. Only the data up to MOD_BUF_SIZE/MOD_SAMPLING_FREQ seconds can be output.
